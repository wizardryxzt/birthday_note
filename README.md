# 生日聚会计划便签

这是一个基于 **React + Vite** 开发的网页应用，用于帮助用户为多位亲友制定生日聚会计划日期。程序支持录入和保存多位亲友的信息，并根据用户输入的“今天日期”和“提前多少天做计划”，自动计算合适的计划日期。

## 项目功能

### 1. 录入并保存多位亲友信息

用户可以录入若干亲友的信息，包括：

- 姓名
- 与用户的关系（如父亲、母亲、爷爷、奶奶、外公、外婆、闺蜜、同学等）
- 出生日期（年、月、日）

程序会将这些亲友信息保存在列表中，支持逐位选择。

### 2. 为指定亲友确定“制定生日聚会计划的日期”

用户可从亲友列表中选择某一位亲友，然后输入：

- 今天的日期
- 需要提前多少天做计划

程序会自动完成以下计算：

- 计算该亲友的下次生日日期
- 计算距离下次生日还有多少天
- 计算“下次生日前 n 天”的原始计划日期

### 3. 自动调整计划日期

若“下次生日前 n 天”的日期是工作日，并且 **不在** 以下特殊日期区间内：

- 5月1日—3日
- 10月1日—7日

则程序会自动将日期调整到最近的周六：

- 周一、周二 → 调整到前一个周六
- 周三、周四、周五 → 调整到后一个周六
- 周六、周日 → 不调整

### 4. 输入校验与特殊情况提示

程序保证运行结果“符合常识”：

- 合法输入会得到合理的计划日期
- 不合法输入会给出明确提醒
- 特殊情况会显示说明或提示信息
- 计划日期不会早于今天
- 计划日期不会出现在生日当天或生日之后
- 如果原计划日期已经不合理，则统一建议：**今天开始准备**

### 5. 结果展示与汇总

在结果显示界面中，程序会展示当前所选亲友的：

- 姓名
- 关系
- 出生日期
- 下次生日日期
- 距离今天还有多少天
- 距离下次生日前 n 天的日期
- 最终建议制定计划的日期

用户可以：

- 重新为当前亲友计算
- 继续为其他亲友制定计划
- 结束并查看最终汇总结果

在结束界面中，程序会汇总显示 **所有已经确定了计划日期的亲友信息**。
